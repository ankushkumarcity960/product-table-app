# Product Table App

## Overview
This project is a React-based web application that displays product data in a table format with editable product titles and infinite scrolling functionality.

## Features
- Fetch product data from DummyJSON API
- Display products in a structured table
- Editable product title field
- Infinite scrolling using Intersection Observer
- Responsive and clean UI

## Tech Stack
- React.js
- JavaScript
- CSS
- Intersection Observer API

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
